GradeTronics - ScanTron Grading App


On-Line Demo

This app was developed by me, Rich Budek, to allow teachers to grade ScanTron forms on their own computer. No grading machine is needed. In addition the software prints an analytical report that lets the teacher know which questions were answered wrong and why.


Click on the top navigation bar for more description or an on-line demo

Description


On the PC version, all the app needs to grade the form is a JPEG image.     The user can mass scan all the forms on a copier, storing the images on a USB drive.  Or, the user can take pictures on their cell phone.

Original Test

The user takes a picture of the test or scans it on a copier's automactic document feeder, the app uses the JPG file.  
 

Graded Test

If an oval is filled in and it should not be, it is marked with an X. If an oval is left blank, it is circled by the app.  
 

Closeup of Grading

The markups can be clearly seen. On the left side, the correct answers are listed along with the points given.  
 

Tabulation at Bottom

The markups can be clearly seen. On the left side, the correct answers are listed along with the points given.  
 

Answers on Right

At the right the correct answers are displayed in red.  
 

Answer Key

The answer key can be simply tapped on the screen or read in from an spread sheet.  
 

Technologies used:
  • Reading JPEG file
  • Computervision to find edge of form
  • Rotating and squaring up an image
  • Scanning an image and locating bubble
  • Determining a human mark from printed one
  • Self-calibrating uniformity matrix
  • MS Visual Studio .NET
  • Compiled with Xamarin for universal mobile apps

The on-line demo uses:
  • Azure VM
  • Convert PC app to cloud app
  • RDP in a Node JS backend
  • Frontend web socket to backend
  • Autolaunch app with RDP

Demonstration of App

The user can simply view the app on-line. It is loaded into a VM (virtual machine) and functions like a cloud app within your browser.  


Alternatively, the user can download a demo PC version of the app. You must have administrative privaledges in order to be able to install the app.
