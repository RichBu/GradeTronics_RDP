Serial Number GCode Generator App


On-Line Demo

While machining parts on a CNC machine, lots of times it becomes necessary to put serial numbers on the parts. Using a CAM program means that the user needs create every single number. This is beneath the diginiy of a human being. This app allows the user to simple select the first digit followed by the second and the app generates the GCode automatically.


Click on the top navigation bar for more description or an on-line demo

Description


The PC versions allows the user to simply select the beginning number of the sequence and the ending sequence. This app then generates the proper GCode program and even puts in the proper comments !!!! First version places the text in two fixed locations on the part. Eventually, the user will be able to select multiple spots.

Original Test

The user takes a picture of the test or scans it on a copier's automactic document feeder, the app uses the JPG file.  
 

Graded Test

If an oval is filled in and it should not be, it is marked with an X. If an oval is left blank, it is circled by the app.
